# Инструкция по тестированию бота

## Подготовка к тестированию

### 1. Убедитесь, что в `.env` файле указаны правильные данные:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_TELEGRAM_ID=ваш_telegram_id  # Для тестирования админки
```

### 2. Тестовые данные уже созданы:

- **10 объявлений** с разными статусами
- **Менеджер 1** (ID: 8280649036): 6 объявлений
  - 3 новых (pending)
  - 2 в работе (accepted, не обработано)
  - 1 обработанное (accepted, обработано)
- **Менеджер 2** (ID: 5745700582): 4 объявления
  - 1 новое (pending)
  - 1 в работе (accepted, не обработано)
  - 1 обработанное (accepted, обработано)
  - 1 отклоненное (rejected)

## Запуск бота для тестирования

```bash
python3 bot/main.py
```

## Основные команды для тестирования

### Для менеджера:
- `/start` - Приветствие
- `/help` - Справка
- `/stats` - Статистика
- `/my_work` - Объявления в работе

### Для админа:
- `/admin` - Дашборд администратора

## Очистка и пересоздание тестовых данных

```bash
# Пересоздать тестовые данные
python3 test_bot_with_fake_data.py

# Очистить только тестовые данные
python3 test_bot_with_fake_data.py clear
```
